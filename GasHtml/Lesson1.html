第一節課：認識 Google Apps Script

暖身問題： 你是否曾經希望能讓試算表自動完成某些重複性的任務，例如快速填寫數據或計算結果？ 今天，我們將學習如何用 Google Apps Script 來實現！

教學目標

  • 了解 Google Apps Script 的基本概念和用途。
  • 學會撰寫並執行第一個簡單的程式。
  • 學會在 Google 試算表中添加自動化功能。

教學內容與操作步驟

1. 撰寫第一個程式


function sayHello() {
    Logger.log("Hello, Google Apps Script!");
}

步驟：

  1. 打開 Google 試算表。
  2. 點擊 擴充功能 > Apps Script。
  3. 將上方的程式碼複製貼上，然後按下工具列的 執行 按鈕。
  4. 查看 日誌輸出，檢查是否看到訊息 "Hello, Google Apps Script!"。
問題：如果你希望程式顯示另一句話，例如 "Hi, Apps Script!"，應該修改哪部分程式碼？

2. 自動填寫試算表數據


function fillSpreadsheet() {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    sheet.getRange(1, 1).setValue("這是自動填寫的內容");
}

步驟：

  1. 將上方程式碼添加到 Apps Script 編輯器中。
  2. 執行程式後，檢查試算表的第一列第一行是否被自動填入。
問題：如果你希望程式填寫的文字是你的名字，應該修改哪部分程式碼？

課程延伸與挑戰

  • 挑戰1： 設計一個程式，將你的名字填入試算表的第二行第一列。
  • 挑戰2： 修改程式，讓它能在多個儲存格中批量填寫數據。
